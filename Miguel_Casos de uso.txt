----------------Texto Casos de Uso---------------
----- Remover Livro -----
:Principal

1) O utilizador clica no botão "Gerir Livros" na janela "Pagina Inicial"
2) O sistema apresenta a janela "Gerir Livros"
3) O utilizador clica no botão "Remover Livro"
4) O sistema apresenta a janela "EliminarFrameLivro"
5) O utilzador clica no botão "Confirmar"
6) O sistema verifica que nao existem resevar pendentes, de seguida redireciona o utilizador para a janela "Gerir Livros"
7) O sistema atualiza a tabela de livros da janela "Gerir Livros"


:Alternativo
5.1) O utilizador clica no botão "Cancelar" na janela "EliminarFrameLivro"
5.1.1) O caso de uso continua no ponto 3 do caminho principal

6.1) O sistema verifica que existem reservas pendentes,de seguida redireciona o utilizador para a janela "Gerir Livros", apresenta um erro


------ Devolver Livro -----
:Principal

1) O utilizador clica no botão "Editar Socio" na janela "GerirSocio"
2) O sistema apresenta a janela "RequisicoesPorSocio"
3) O utilizador clica no botão "Devolver" na janela "RequisiçõesPorSocio"
4) O sistema apresenta a janela "FrameValoraPagar"
5) O utilzador clica no botão "Pago"
6) O sistema atualiza a coluna "Data Devolução" da janela "RequisiçõesPorSocio"

:Alternativo
5.1) O utilizador clica no botão "Não Pago" na janela "FrameValoraPagar"
5.1.1) O caso de uso continua no ponto 1 do caminho principal



------ Pesquisar Livro -----
:Principal

1) O utilizador clica no botão "Gerir Livros" na janela "Pagina Inicial"
2) O sistema apresenta a janela "Gerir Livros"
3) O utilizador preenche o "Procurar Livro"
4) O sistema verifica quais livros pode apresentar e atualiza a tabela de livros da janela "Gerir Livros"

:Alternativo
4.1) O sistema nao conseguir verificar quais livros pode apresentar e volta a janela "Gerir Livros", e apresenta um erro

------ Requisitar Livro -----
:Principal

1) O utilizador clica no botão "Requisitar" na janela "Requisitar Livros"
2) O sistema calcula as datas de levantamento, devoluçao e apresenta a janela "ConfirmaRequisição"
3) O utilizador valida os dados e clica no botão "Guardar"
4) O sistema verifica que o livro esta disponivel e volta a janela "RequisicoesPorSocio"
5) O sistema atualiza a tabela de livros da janela "RequisicoesPorSocio"

:Alternativo
2.1) O sistema nao consegue calcular as datas de levantamento e devoluçao, de seguida mantem o utilizador para a janela "Requisitar Livros", e apresenta um erro 

3.1) O utilizador valida os dados e clica no botão "Cancelar"
3.1.1) o sistema apresenta a janela "ConfirmarCancelamento"
3.1.2) O utilizador clica no botão "Sim" na janela "ConfirmarCancelamento"
3.1.1) O caso de uso volta ao ponto 1 do caminho principal

3.1.2.1) O utilizador clica no botão "Não" na janela "ConfirmarCancelamento"
3.1.2.2) O caso de uso volta ao ponto 2 do caminho principal

4.1) O sistema verifica que o livro nao esta disponivel, volta ao janela "Requisitar Livros" e apresenta um erro


------ Reservar Livro -----
:Principal

1) O utilizador clica no botão "Reservar" na janela "Requisitar Livros"
2) O sistema sistema calcula o numero de Reservas e apresenta a janela "ConfirmaReserva"
3) O utilizador valida os dados e clica no botão "Guardar"
4) O sistema verifica que o livro esta indisponivel e volta ao janela "Requisitar Livros"
5) O sistema atualiza a tabela de livros da janela "Requisitar Livros"

:Alternativo
2.1) O sistema nao consegue calcular o numero de reservas, de seguida mantem o utilizador na janela "Requisitar Livros", e apresenta um erro 

3.1) O utilizador valida os dados e clica no botão "Cancelar"
3.1.1) o sistema apresenta a janela "ConfirmarCancelamento"
3.1.2) O utilizador clica no botão "Sim" na janela "ConfirmarCancelamento"
3.1.3) O caso de uso volta ao ponto 1 do caminho principal

3.1.2.1) O utilizador clica no botão "Não" na janela "ConfirmarCancelamento"
3.1.2.2) O caso de uso volta ao ponto 2 do caminho principal

4.1) O sistema verifica que o livro esta disponivel, volta ao janela "Requisitar Livros" e apresenta um erro
